Not all RSS feed readers/viewers are able to interpret HTML tags. When reading arXiv feeds, this leads to a very cluttered feed when author lists are long:

Before After

This site dynamically generates HTML-stripped versions of the arXiv RSS feeds. To use the cleaned versions, use the links below in your RSS reader instead of the official arXiv ones.

The code for the Google App engine is available on GitHub. Please report any issues there, and please do contribute to the code if you have ideas to improve it! I've only included the Astrophysics feeds so far, but adding all the other would be trivial, so contributions are welcome!

Disclaimer: This website is provided as a free service, and is still experimental! I cannot make any guarantees that valid data will not be removed from the feeds.
