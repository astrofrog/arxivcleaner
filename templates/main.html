About

Not all RSS feed readers/viewers are able to interpret HTML tags. For example, Google Reader does not interpret HTML tags in the author section. When reading ArXiv feeds, this leads to a very cluttered feed when author lists are long.

This site dynamically generates HTML-stripped versions of the ArXiv RSS feeds. To use them cleaned versions, follow the links below, and use the resulting URLs in your RSS reader instead of the official ArXiv ones.

Disclaimer: This website is provided as a free service, and is still experimental. I cannot make any guarantees that valid data will not be removed from the feeds.

The code for the Google App engine is available on GitHub. Please report any issues there, and please do contribute to the code if you have ideas to improve it!
